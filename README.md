# RM. Pondok Marisa - Sistem POS Web

Sistem Point of Sale berbasis web untuk RM. Pondok Marisa.

## Struktur File

| File | Keterangan |
|------|-----------|
| index.html | Dashboard Admin |
| kasir.html | Halaman Kasir / POS |
| produk.html | Manajemen Produk |
| transaksi.html | Riwayat Transaksi |
| laporan.html | Laporan Penjualan |
| pengaturan.html | Pengaturan Sistem |
| menu-hub.html | Hub Menu Digital & QR Code |
| menu.html | Menu Digital (tampilan pelanggan) |
| menu-admin.html | Admin Kelola Menu & Pesanan |
| qrcode.html | Generator QR Code |
| admin-style.css | Stylesheet halaman Admin |
| admin-app.js | Script utama Admin (Firebase + localStorage) |
| menu-style.css | Stylesheet Menu Digital |
| menu-app.js | Script Menu Digital |
| menu.json | Data menu default (JSON) |
